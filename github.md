# Quick Setup

## Create new repo from command line

	echo "# notes" >>  README.md
	git init
	git add README.md
	git commit -m "first commit"
	git branch -M main
	git remote add origin https://github.com/[repo][project.git]
	git push -u origin main

## Or push repository from command line

	git remote add origin https://github.com/[repo][project.git]
	git branch -M main
	git push -u origin main
